hexo 添加live2D

发表于2025-01-29|更新于2025-01-29|技术
|浏览量:

添加插件：

npm install --save hexo-helper-live2d

然后在hexo的_config.yml中增加live2d的配置

支持很多种模型： 模型预览

用哪个模型需要提前安装:

npm install 模型名称

安装模型之后，需要修改安装的模型名称

TODO： 可替换为加强版live2D，可换装、文字，再接入deepsek之类的大模型效果更佳
